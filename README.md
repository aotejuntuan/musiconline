# MusicOnline

简介：一个基于PHP的B/S在线音乐网站，用户可以保存自己的歌单在服务器上、上传自己的歌曲、搜索想要的歌曲、下载喜欢的歌曲以及简单的用户个人资料更新管理等。

技术栈：Apache/PHP/MySql、jQuery、Bootstrap以及Ajax。
